# 所需python包

- flask
- flask-restful
- requests

## 运行环境

- python≥3.5

## 运行
```shell
> git clone https://github.com/NomadJohn/CET4
> cd CET4
> pip3 install flask flask_restful requests
> python3 app.py
```

访问

- http://localhost:8080
